## Story
**As a** visitante da LP,
**I want** que elementos se animem suavemente conforme eu rolo a página,
**so that** a experiência seja envolvente e moderna sem ser intrusiva.

## Acceptance Criteria
1. GSAP e ScrollTrigger carregados via CDN no `index.html` (antes do `main.js`)
2. Todas as seções têm animação de entrada (fade-in + slide-up) ativada por scroll — **não automática ao carregar**
3. Números/estatísticas na seção Solução animam contagem do 0 ao valor final ao entrar na viewport
4. Countdown timer funcional em `js/main.js` com sessionStorage para persistir tempo restante por sessão
5. FAQ accordion funcional via JS: clicar na pergunta abre/fecha a resposta com animação suave (GSAP)
6. Botão WhatsApp flutuante aparece após o usuário rolar 300px, usa `WHATSAPP_NUMBER` do `config.js`
7. Animações completamente desabilitadas quando `prefers-reduced-motion: reduce` está ativo
8. Zero erros no console do browser em mobile (375px) e desktop (1280px)

### Countdown com sessionStorage
```javascript
function initCountdown() {
  const DURATION = 15 * 60; // 15 minutos em segundos
  const key = 'lp_countdown_end';

  let endTime = sessionStorage.getItem(key);
  if (!endTime) {
    endTime = Date.now() + DURATION * 1000;
    sessionStorage.setItem(key, endTime);
  }

  setInterval(() => {
    const remaining = Math.max(0, Math.floor((endTime - Date.now()) / 1000));
    const mm = String(Math.floor(remaining / 60)).padStart(2, '0');
    const ss = String(remaining % 60).padStart(2, '0');
    document.querySelectorAll('.countdown-display').forEach(el => {
      el.textContent = `${mm}:${ss}`;
    });
  }, 1000);
}
```
