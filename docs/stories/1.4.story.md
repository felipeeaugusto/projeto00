### Deploy via Vercel drag-and-drop
1. Acessar https://vercel.com e fazer login
2. Clicar em "Add New" → "Project"
3. Selecionar "Browse" → arrastar a pasta `landing-page-dr-julia/`
4. Framework: "Other" (HTML estático)
5. Build Command: deixar vazio
6. Output Directory: `.` (raiz)
7. Clicar "Deploy"

### Deploy via Vercel CLI
```bash
npm install -g vercel
cd C:/Users/felip/projeto00/packages/landing-page-dr-julia
vercel --prod
```
