## Story
**As a** visitante da LP,
**I want** que a página tenha visual acolhedor, profissional e responsivo,
**so that** eu tenha uma boa experiência tanto no celular quanto no desktop.

## Acceptance Criteria
1. `css/style.css` criado com CSS variables: `--verde: #03bb85`, `--verde-claro: #f0fdf9`, fontes e espaçamentos
2. Poppins carregada via `@font-face` local (`assets/fonts/Poppins-Bold.ttf`, `Poppins-SemiBold.ttf`, `Poppins-Regular.ttf`)
3. Layout mobile-first com breakpoints: 375px (base), 768px (tablet), 1280px (desktop)
4. Hero: foto Dra. Julia + mockup lado a lado no desktop / empilhados no mobile (foto topo, mockup abaixo)
5. Cards de produto: Combo visualmente maior, borda verde #03bb85, badge "Mais Escolhido" em destaque; E-book discreto e menor
6. Seção Bônus em grid: 3 colunas (1280px) / 2 colunas (768px) / 1 coluna (375px)
7. Countdown timer placeholder com estilo de destaque posicionado acima do CTA principal
8. Selo de garantia com borda arredondada, fundo verde claro e ícone de escudo
9. PageSpeed Insights mobile >= 85 (com imagens otimizadas)
10. Nenhum layout quebrado em viewport 375px, 768px e 1280px

### Paleta de cores (DS.yaml v2.0)

```css
:root {
  --verde: #03bb85;
  --verde-claro: #f0fdf9;
  --branco: #ffffff;
  --texto: #1a1a1a;
  --texto-secundario: #555555;
  --fonte-principal: 'Poppins', sans-serif;
}
```

### @font-face — localização das fontes
```css
@font-face {
  font-family: 'Poppins';
  src: url('../assets/fonts/Poppins-Regular.ttf') format('truetype');
  font-weight: 400;
  font-display: swap;
}
@font-face {
  font-family: 'Poppins';
  src: url('../assets/fonts/Poppins-SemiBold.ttf') format('truetype');
  font-weight: 600;
  font-display: swap;
}
@font-face {
  font-family: 'Poppins';
  src: url('../assets/fonts/Poppins-Bold.ttf') format('truetype');
  font-weight: 700;
  font-display: swap;
}
```

### Estratégia mobile-first
- Escrever TODOS os estilos base para 375px
- Usar APENAS `min-width` nos media queries (nunca `max-width`)
- Ordem: base → `@media (min-width: 768px)` → `@media (min-width: 1280px)`

### Paralelismo com Story 1.1
Esta story pode ser desenvolvida em paralelo com a 1.1. O CSS pode ser escrito independentemente — quando o HTML da 1.1 estiver pronto, aplicar e testar juntos.
