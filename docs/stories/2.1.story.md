## Story
**As a** Felipe,
**I want** um script Python que receba um elemento PNG já baixado do ChatGPT e monte o post final com fundo de cor exata, texto em Poppins, assinatura e logo,
**so that** eu possa validar a qualidade da composição manualmente antes de automatizar o processo completo.

## Acceptance Criteria
1. Script roda com `python compositor.py --ajuda` e exibe todos os parâmetros disponíveis
2. Fundo gerado por código com cor exata `RGB(3, 187, 133)` (#03bb85) — verificável via color picker
3. Elemento PNG fornecido pelo Felipe é posicionado centralizado sobre o fundo
4. Texto em Poppins ExtraBold aplicado com cor branca (#FFFFFF) sobre fundo verde
5. Assinatura `@drjuliaresende` posicionada bottom-right com 24px de margem, fonte Poppins Regular, cor branca
6. Logo do abraço (`logo-abraco-branco.png`) aplicado sobre o fundo
7. Output salvo no caminho informado em `--saida` como PNG 1080x1080px RGB 72dpi
8. 3 posts de teste compostos e validados pelo Felipe:
   - Teste 1 — Formato VE (Visual Educativo, fundo verde)
   - Teste 2 — Formato QC (Quote Card, fundo branco)
   - Teste 3 — Formato MT (Meme/Tirinha, fundo branco)
9. Script NÃO chama DALL-E API — Felipe fornece o elemento manualmente
